# 📊 Epic 7: Analytics — Dashboard Báo cáo

> Hệ thống báo cáo thông minh giúp Admin nắm bắt tình hình kinh doanh nhanh chóng.

---

## 🏁 Mục tiêu (Goal)
Số hóa các chỉ số kinh doanh thay vì Admin phải tự tính toán thủ công từ danh sách đơn hàng.

---

## 📦 Phạm vi (Scope)

### Trong phạm vi (In Scope):
- [ ] API lấy thống kê tổng quan (Doanh thu, Đơn hàng).
- [ ] API lấy danh sách Top sản phẩm bán chạy.
- [ ] Trang Dashboard Admin với biểu đồ và số liệu.

### Ngoài phạm vi (Out of Scope):
- Xuất báo cáo ra file Excel/PDF (Phase 3).
- Dự báo doanh thu bằng AI.

---

## ✅ Acceptance Criteria
- [ ] Dashboard hiển thị đúng tổng doanh thu từ các đơn hàng `DONE`.
- [ ] Biểu đồ hiển thị sự thay đổi doanh thu theo thời gian.
- [ ] Admin thấy được sản phẩm nào đang mang lại doanh thu cao nhất.
